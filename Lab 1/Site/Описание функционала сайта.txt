Сайт для пользователя открывается на странице каталога с возможностью переключения на страницу сервисных услуг.
Контакты представляют собой всплывающее окно, открывающееся при нажатии на соответствующий элемент.
Элементы вышеперечисленных функций находятся в шапке сайта.

Для поиска автомобилей поддерживаются следующие фильтры:
  1) Поиск по названию автомобиля
  2) Фильтр по цене
  3) Фильтр по году выпуска автомобиля
  4) Фильтр по модели автомобиля

Страница сервисных услуг поддерживает функцию поиска по названию услуги.

При выборе интересующего автомобиля или услуги пользователь может оставить заявку, оставив необходимую информацию о себе.

Страница для менеджера представляет собой табличное представление необходимой информации о заявках, клиентах, автомобилях или услугах.
Все перечисленные поля представлены в шапке сайта с возможностью переключения между ними по нажатию на соответствующий элемент.
